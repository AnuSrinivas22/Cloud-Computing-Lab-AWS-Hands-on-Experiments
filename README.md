# Cloud-Computing-Lab-AWS-Hands-on-Experiments
A collection of cloud computing lab experiments including AWS EC2 setup, IAM design, virtualization, security, and networking concepts.

# ☁️ Cloud Computing Lab – AWS Hands-on Experiments

## 📌 Overview

This repository contains all the lab experiments performed as part of the Cloud Computing course. The experiments focus on practical implementation of cloud concepts using AWS and virtualization tools.

---

## 🧪 List of Experiments

1. Launch and Configuration on Amazon EC2
2. Creation of Linux EC2 Instance on AWS
3. ISO Mounting and OS Installation in Virtual Machine
4. Understanding and Designing a Virtual Machine
5. Security, Access, and URL-Based Object Management
6. Concepts and Configuration of Static Website Hosting
7. Designing IAM for a Small Organization
8. Fundamentals and Creation of Relational Database (RDS)
9. End-to-End Scenario – Designing a Cloud Architecture

---

## ⚙️ Tools & Technologies Used

* Amazon Web Services (AWS)
* EC2 (Elastic Compute Cloud)
* IAM (Identity and Access Management)
* RDS (Relational Database Service)
* VirtualBox / VMware
* Linux (Ubuntu/CentOS)

---

## 🎯 Objectives

* Understand cloud computing fundamentals
* Gain hands-on experience with AWS services
* Learn virtualization and system setup
* Implement security and access control using IAM
* Deploy and manage cloud-based applications

---

## 📂 Folder Structure

Each experiment is organized into separate folders containing:

* Documentation (steps, explanation)
* Screenshots (if applicable)
* Configuration files

---

## 🚀 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/cloud-computing-lab.git
   ```
2. Open individual experiment folders
3. Follow the steps provided in each experiment
